# Hola, soy Marco Garcia 👋

Soy desarrollador full stack con base en Chiclayo, Perú. Trabajo en el área de TI de la **USAT (Universidad Católica Santo Toribio de Mogrovejo)**, donde desarrollo y mantengo sistemas de gestión académica. También me dedico a crear contenido educativo en programación y a compartir lo que voy aprendiendo.

Me muevo cómodo entre el backend legacy y las arquitecturas modernas — desde stored procedures en SQL Server hasta microservicios con .NET 8 y Docker.
